# Página Web Turística - Maripí Ecoturismo

## Descripción
Página web interactiva diseñada para promocionar el ecoturismo en Maripí, Boyacá, destacando las oportunidades económicas para familias rurales con un diseño moderno y efectos neón.

## Características
- ✨ Efectos neón vibrantes en títulos, botones y navegación
- 🌿 Enfoque en ecoturismo sostenible
- 💼 Sección dedicada a oportunidades económicas rurales
- 📱 Diseño responsive (móvil y desktop)
- 🎨 Interactividad completa con JavaScript
- 📧 Formulario de contacto funcional
- 🖼️ Galería de imágenes interactiva

## Estructura del Proyecto
```
maripi-ecoturismo/
├── index.html          # Página principal
├── css/
│   └── styles.css      # Estilos CSS con efectos neón
├── js/
│   └── script.js       # JavaScript para interactividad
├── images/             # Imágenes del proyecto
│   ├── hero-landscape.jpg
│   ├── maripi-town.jpg
│   └── ecoturismo.jpg
├── videos/             # Carpeta para videos (vacía)
└── README.md           # Este archivo
```

## Cómo usar

### Opción 1: Abrir directamente
1. Descomprimir el archivo
2. Abrir `index.html` en cualquier navegador web moderno

### Opción 2: Servidor local (recomendado)
1. Descomprimir el archivo
2. Abrir terminal en la carpeta del proyecto
3. Ejecutar: `python -m http.server 8000` (Python 3) o `python -m SimpleHTTPServer 8000` (Python 2)
4. Abrir navegador en: `http://localhost:8000`

## Personalización
Para personalizar la página:
1. **Colores**: Modificar variables CSS en `:root` en `styles.css`
2. **Contenido**: Editar texto en `index.html`
3. **Imágenes**: Reemplazar archivos en carpeta `images/`
4. **Efectos**: Ajustar animaciones en `styles.css` y `script.js`
